== Description ==

This plugin was inspired by Matt Mullenweg's <a href="https://wordpress.org/plugins/hello-dolly/">Hello Dolly</a> plugin. This plugin is simple and rotates tips for WordPress users in the top right corner of the WordPress admin menu, right near the Help and Screen Options tabs on every page. Tips added by Nile Flores of <a href="http://blondish.net">Blondish.net</a>.

== Installation ==

1. Unzip file
2. Upload folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
